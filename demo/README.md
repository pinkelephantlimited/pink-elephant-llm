# Pink Elephant LLM 1.3B Demo

A Gradio chat interface for the **Pink Elephant LLM 1.3B** model.

Model: [pinkelephantlimited/pink-elephant-llm-1.3b](https://huggingface.co/pinkelephantlimited/pink-elephant-llm-1.3b)

## Usage

Type a message and the model will generate a response. The conversation history is maintained for context.
